Inéquation du second degré

Pour travailler votre inéquation vous allez devoir completer un formulaire avec les données de votre inéquation
Nous savons qu'une inéquation du second degré est de forme:
ax2 + bx + c >(ou <,≥,≤) c avec a different de 0 car si a est égal à 0 alors ce n'est plus une inéquation du second degré mais du première degré (car 0x sera toujours égal à 0 comme il est dit tout nombre multiplier par 0 égal toujours à 0)
Donc pour calculer nous aurons besoin de ces trois éléments a,b,c qui ont chacun une case reserver pour qu'il y soit completer par exemple pour l'inéquation
2x2 + 4x + 5 > 0 la case a = 2 , b = 4 et c = 5
Puis à la fin il faudra cocher le signe de comparaison
> : Supérieur
< : inférieur
≤ : inférieur ou égal
≥: supérieur ou égal
NB : Si vous avez une fraction dans votre inéquation il faudra seulement la diviser puis completer le quotient dans la case correspondente(exemple pour 1/3 on completera 0,3333...)
